Jwane Designs is a high-performance, edge-first visual architecture platform built for the next generation of top-tier creators. This isn't just a portfolio; it's a dynamic Content Management System (CMS) engineered for speed, conversion, and surgical precision.

## ⚡ The Tech Stack

*   **Logic**: [Vite](https://vitejs.dev/) + [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Framer Motion](https://www.framer.com/motion/)
*   **Backend**: [Supabase](https://supabase.com/) (DB, Auth, Storage)
*   **Aesthetics**: Glassmorphism / Cyber-Cyan / High-Fidelity Motion

## 🎯 Features

### 1. 'Elite' Admin Dashboard
Located at `/muhee`, this secure portal allows the brand owner to manage the entire ecosystem without touching a single line of code.
-   **Portfolio Engine**: Real-time CRUD for thumbnails with direct-to-edge cloud uploads.
*   **Social Proof Roster**: Dynamic management of creator testimonials and influence metrics.

### 2. Edge-Hydrated Storefront
The public site is entirely dynamic. Your latest viral assets and creator feedback are streamed directly from the Supabase edge, ensuring absolute data parity and zero rebuild latency.

### 3. Surgical Animations
Integrated Framer Motion pipelines provide micro-interactions that wow users at first glance.

## 🛠️ Installation & Setup

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/unnamed-yet.git
    cd unnamed-yet
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Environment**:
    Create a `.env` file in the root and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Launch the Engine**:
    ```bash
    npm run dev
    ```
